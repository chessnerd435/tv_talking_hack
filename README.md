# TV Talking Hack

This is a fun tool to control any lg webos TV from your command line. You can make the TV "speak" messages using its text-to-speech feature, or turn it on/off!

## Prerequisites

first install node


- **Mac/Windows**: Download and install from [nodejs.org](https://nodejs.org/).
- **Linux (Ubuntu/Debian)**:
  ```bash
  sudo apt update
  sudo apt install nodejs npm
  ```
 
```bash
cd /your/file/location/
```  
next you need to cd into the folder that you have the files that you downloaded in


## How to Install

then you need to install node_modules inside the folder

```bash
npm install
```

## How to Use

Run the script with Node:

```bash
node tv_say.js put anything you want to say here 
```

The script is interactive. It will ask you for:
1.  **IP Address**: The IP address of the TV on your network.
